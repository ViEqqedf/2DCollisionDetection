\doxysubsection{Detailed Description}
A standard toggle that has an on / off state. 

The toggle component is a \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_selectable}{Selectable}} that controls a child graphic which displays the on / off state. When a toggle event occurs a callback is sent to any registered listeners of UI.\+Toggle.\+\_\+on\+Value\+Changed.
