\doxysubsection{Detailed Description}
Base class for any Ray\+Caster. 

A Raycaster is responsible for raycasting against scene elements to determine if the cursor is over them. Default Raycasters include \mbox{\hyperlink{class_unity_engine_1_1_event_systems_1_1_physics_raycaster}{Physics\+Raycaster}}, \mbox{\hyperlink{class_unity_engine_1_1_event_systems_1_1_physics2_d_raycaster}{Physics2\+DRaycaster}}, Graphic\+Raycaster. Custom raycasters can be added by extending this class.
