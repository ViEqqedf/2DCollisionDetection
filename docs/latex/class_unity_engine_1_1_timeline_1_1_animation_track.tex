Creates an Animation\+Clip that stores the data for an infinite track. 

If an infinite\+Clip already exists, this method produces no result, even if you provide a different value for infinite\+Clip\+Name. 

This method can\textquotesingle{}t create an infinite clip for an \mbox{\hyperlink{class_unity_engine_1_1_timeline_1_1_animation_track}{Animation\+Track}} that contains one or more \mbox{\hyperlink{namespace_unity_engine_1_1_timeline}{Timeline}} clips. Use \mbox{\hyperlink{class_unity_engine_1_1_timeline_1_1_animation_track_aa1135c27853312d233b0b711e4b849c3}{Animation\+Track.\+in\+Clip\+Mode}} to determine whether it is possible to create an infinite clip on an \mbox{\hyperlink{class_unity_engine_1_1_timeline_1_1_animation_track}{Animation\+Track}}. 

When used from the editor, this method attempts to save the created infinite clip to the \mbox{\hyperlink{class_unity_engine_1_1_timeline_1_1_timeline_asset}{Timeline\+Asset}}. The \mbox{\hyperlink{class_unity_engine_1_1_timeline_1_1_timeline_asset}{Timeline\+Asset}} must already exist in the Asset\+Database to save the infinite clip. If the \mbox{\hyperlink{class_unity_engine_1_1_timeline_1_1_timeline_asset}{Timeline\+Asset}} does not exist, the infinite clip is still created but it is not saved.
