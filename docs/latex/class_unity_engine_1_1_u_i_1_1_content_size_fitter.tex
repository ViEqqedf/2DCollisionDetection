\doxysubsection{Detailed Description}
Resizes a Rect\+Transform to fit the size of its content. 

The \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_content_size_fitter}{Content\+Size\+Fitter}} can be used on Game\+Objects that have one or more \mbox{\hyperlink{interface_unity_engine_1_1_u_i_1_1_i_layout_element}{ILayout\+Element}} components, such as \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_text}{Text}}, \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_image}{Image}}, \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_horizontal_layout_group}{Horizontal\+Layout\+Group}}, \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_vertical_layout_group}{Vertical\+Layout\+Group}}, and \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_grid_layout_group}{Grid\+Layout\+Group}}.
