\doxysubsection{Detailed Description}
Base class for a timeline action. Inherit from this class to make an action on a timeline after a menu click and/or a key shortcut. 

To add an action as a menu item in the \mbox{\hyperlink{namespace_unity_editor_1_1_timeline}{Timeline}} context menu, add \mbox{\hyperlink{class_unity_editor_1_1_timeline_1_1_actions_1_1_menu_entry_attribute}{Menu\+Entry\+Attribute}} on the action class. To make an action to react to a shortcut, use the Shortcut Manager API with \mbox{\hyperlink{class_unity_editor_1_1_timeline_1_1_actions_1_1_timeline_shortcut_attribute}{Timeline\+Shortcut\+Attribute}}. \begin{DoxySeeAlso}{See also}
Unity\+Editor.\+Shortcut\+Management.\+Shortcut\+Attribute, \mbox{\hyperlink{class_unity_editor_1_1_timeline_1_1_actions_1_1_active_in_mode_attribute}{Active\+In\+Mode\+Attribute}}


\end{DoxySeeAlso}


Simple \mbox{\hyperlink{namespace_unity_editor_1_1_timeline}{Timeline}} Action example (with context menu and shortcut support).
