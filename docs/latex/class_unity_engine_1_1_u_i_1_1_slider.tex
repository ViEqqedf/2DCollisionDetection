\doxysubsection{Detailed Description}
A standard slider that can be moved between a minimum and maximum value. 

The slider component is a \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_selectable}{Selectable}} that controls a fill, a handle, or both. The fill, when used, spans from the minimum value to the current value while the handle, when used, follow the current value. The anchors of the fill and handle Rect\+Transforms are driven by the \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_slider}{Slider}}. The fill and handle can be direct children of the Game\+Object with the \mbox{\hyperlink{class_unity_engine_1_1_u_i_1_1_slider}{Slider}}, or intermediary Rect\+Transforms can be placed in between for additional control. When a change to the slider value occurs, a callback is sent to any registered listeners of UI.\+Slider.\+on\+Value\+Changed.
